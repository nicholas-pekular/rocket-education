# rocket-education
code snippets for educational model rocket stuff

# prerequisites

```bash
# install capture and encoder bins
sudo apt install -y python3-picamera2
sudo apt install ffmpeg

# optional: install play playback bins (if you want to playback on the device itself)
sudo apt install vlc
```

# run it

```bash
# capture a video
python capture_video.py

# playback (local)
vlc video_xxxx.mp4 
```
