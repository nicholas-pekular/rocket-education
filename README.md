# rocket-education
code snippets for educational model rocket stuff

## Services

### Video Capture Service

A containerized video capture service for Raspberry Pi Zero 2 W with IMX camera, controlled via FastAPI.

See [video-capture-service/README.md](video-capture-service/README.md) for detailed documentation.

**Quick Start:**
```bash
cd video-capture-service
docker-compose up -d --build
```
